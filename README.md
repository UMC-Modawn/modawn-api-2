## Setting

* installation
```bash
$ yarn add
```

* .env
```bash
$ cp .env.example .env
```

* ormconfig.json
```bash
$ cp ormconfig.example.json ormconfig.json
```

## Running the app

```bash
$ yarn run start:dev
```

## Run Apidoc

```bash
# apidoc build
$ yarn run apidoc:build

# apidoc start
$ yarn run apidoc:start
```
